---
title: "Получить пользователя по ID"
api: "GET http://207.154.237.52:8000/user/get-user/{id}"
description: "Получение данных конкретного пользователя по его UUID"
---

## Параметры пути

<ParamField path="id" type="string" required>
  Уникальный идентификатор пользователя (UUID)
</ParamField>

## Ответ

<ResponseField name="id" type="string">
  Уникальный идентификатор пользователя (UUID)
</ResponseField>

<ResponseField name="email" type="string">
  Email пользователя
</ResponseField>

<ResponseField name="nickname" type="string">
  Никнейм пользователя
</ResponseField>

<ResponseField name="username" type="string">
  Имя пользователя
</ResponseField>

<ResponseField name="status" type="string">
  Роль пользователя (`user`, `admin` и т.д.)
</ResponseField>

<ResponseField name="user_avatar" type="string">
  Путь к аватару пользователя
</ResponseField>

<ResponseField name="user_banner" type="string">
  Путь к баннеру пользователя
</ResponseField>

<ResponseField name="password" type="string">
  Хешированный пароль (bcrypt)
</ResponseField>

<RequestExample>
```bash cURL
curl -X 'GET' \
  'http://207.154.237.52:8000/user/get-user/dfaad63b-4a3f-4bf0-9ed3-e5d007e9e321' \
  -H 'accept: application/json'
```

```javascript JavaScript
const response = await fetch(
```

```python Python
import requests
```

## Параметры запроса

<ParamField query="user_id" type="string" required>
  UUID пользователя
</ParamField>

## Ответ

Возвращает файл изображения (binary).

<RequestExample>
```bash cURL
curl -X 'GET' \
  'http://207.154.237.52:8000/user/get-user-banner?user_id=dfaad63b-4a3f-4bf0-9ed3-e5d007e9e321' \
  -H 'accept: application/json'
```

```javascript JavaScript
const response = await fetch(
  "http://207.154.237.52:8000/user/get-user-banner?user_id=dfaad63b-4a3f-4bf0-9ed3-e5d007e9e321"
);
const blob = await response.blob();
const imageUrl = URL.createObjectURL(blob);
```

```python Python
import requests

response = requests.get(
    "http://207.154.237.52:8000/user/get-user-banner",
    params={"user_id": "dfaad63b-4a3f-4bf0-9ed3-e5d007e9e321"}
)
with open("banner.png", "wb") as f:
    f.write(response.content)
```
</RequestExample>

---
title: "Обновить аватар"
api: "PUT http://207.154.237.52:8000/user/update-my-avatar"
description: "Загрузка нового аватара текущего пользователя"
---

## Авторизация

<ParamField header="Authorization" type="string" required>
  Bearer JWT-токен. Пример: `Bearer eyJhbGci...`
</ParamField>

## Тело запроса

<Note>
  Запрос отправляется как `multipart/form-data`.
</Note>

<ParamField body="file" type="file" required>
  Файл изображения (JPEG, PNG и т.д.)
</ParamField>

## Ответ

<ResponseField name="message" type="string">
  Сообщение об успешном обновлении
</ResponseField>

<RequestExample>
```bash cURL
curl -X 'PUT' \
  'http://207.154.237.52:8000/user/update-my-avatar' \
  -H 'accept: application/json' \
  -H 'Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...' \
  -H 'Content-Type: multipart/form-data' \
  -F 'file=@avatar.jpg;type=image/jpeg'
```

```javascript JavaScript
const formData = new FormData();
formData.append("file", fileInput.files[0]);

const response = await fetch(
  "http://207.154.237.52:8000/user/update-my-avatar",
  {
    method: "PUT",
    headers: {
      Accept: "application/json",
      Authorization: "Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    },
    body: formData,
  }
);
const result = await response.json();
```

```python Python
import requests

with open("avatar.jpg", "rb") as f:
    response = requests.put(
        "http://207.154.237.52:8000/user/update-my-avatar",
        headers={"Authorization": "Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."},
        files={"file": ("avatar.jpg", f, "image/jpeg")}
    )
result = response.json()
```

## Авторизация

<ParamField header="Authorization" type="string" required>
  Bearer JWT-токен. Пример: `Bearer eyJhbGci...`
</ParamField>

## Параметры запроса

<ParamField query="nickname" type="string" required>
  Новый никнейм пользователя
</ParamField>

## Ответ

<ResponseField name="message" type="string">
  Сообщение об успешном обновлении
</ResponseField>

<RequestExample>
```bash cURL
curl -X 'PUT' \
  'http://207.154.237.52:8000/user/update-my-nickname?nickname=adams' \
  -H 'accept: application/json' \
  -H 'Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...'
```

import requests

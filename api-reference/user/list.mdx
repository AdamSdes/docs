---
title: "Получить пользователей"
api: "GET http://207.154.237.52:8000/user/"
description: "Получение списка пользователей с пагинацией"
---

## Параметры запроса

<ParamField query="page" type="integer" default="1">
  Номер страницы для пагинации
</ParamField>

<ParamField query="limit" type="integer" default="5">
  Количество записей на странице
</ParamField>

## Ответ

Возвращает массив объектов пользователей.

<ResponseField name="id" type="string">
  Уникальный идентификатор пользователя (UUID)
</ResponseField>

<ResponseField name="email" type="string">
  Email пользователя
</ResponseField>

<ResponseField name="nickname" type="string">
  Никнейм пользователя
</ResponseField>

<ResponseField name="username" type="string">
  Имя пользователя
</ResponseField>

<ResponseField name="status" type="string">
  Роль пользователя (`user`, `admin` и т.д.)
</ResponseField>

<ResponseField name="user_avatar" type="string">
  Путь к аватару пользователя
</ResponseField>

<ResponseField name="user_banner" type="string">
  Путь к баннеру пользователя
</ResponseField>

<ResponseField name="password" type="string">
  Хешированный пароль (bcrypt)
</ResponseField>

<RequestExample>
```bash cURL
curl -X 'GET' \
  'http://207.154.237.52:8000/user/?page=1&limit=5' \
  -H 'accept: application/json'
```

```javascript JavaScript
const response = await fetch(
  "http://207.154.237.52:8000/user/?page=1&limit=5",
  {
    headers: {
      Accept: "application/json",
    },
  }
);
const users = await response.json();
```

```python Python
import requests

response = requests.get(
    "http://207.154.237.52:8000/user/",
    params={"page": 1, "limit": 5},
    headers={"Accept": "application/json"}
)
users = response.json()
```
</RequestExample>

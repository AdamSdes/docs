---
title: "Добавить аниме в список"
api: "PUT http://207.154.237.52:8000/anime_save_list/put_anime_id_in_list/{list_name}"
description: "Добавление аниме по ID в указанный список авторизованного пользователя"
---

## Параметры пути

<ParamField path="list_name" type="string" required>
  Название списка. Допустимые значения: `Watching`, `Completed`, `On Hold`, `Dropped`, `Plan to Watch`
</ParamField>

## Параметры запроса

<ParamField query="anime_id" type="string" required>
  ID аниме для добавления в список (Shikimori ID)
</ParamField>

## Заголовки

<ParamField header="Authorization" type="string" required>
  Bearer-токен для аутентификации
</ParamField>

## Ответ

<ResponseField name="id" type="string">
  UUID списка
</ResponseField>

<ResponseField name="user_id" type="string">
  UUID пользователя-владельца
</ResponseField>

<ResponseField name="list_name" type="string">
  Название списка
</ResponseField>

<ResponseField name="anime_ids" type="array">
  Обновлённый массив ID аниме в списке
</ResponseField>

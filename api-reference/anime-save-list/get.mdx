description: "Получение списка сохранённых аниме по названию списка и ID пользователя"

## Параметры пути

<ParamField path="list_name" type="string" required>
  Название списка. Допустимые значения: `Watching`, `Completed`, `On Hold`, `Dropped`, `Plan to Watch`
</ParamField>

<ParamField path="user_id" type="string" required>
  UUID пользователя
</ParamField>

## Ответ

<ResponseField name="id" type="string">
  UUID списка
</ResponseField>

<ResponseField name="user_id" type="string">
  UUID пользователя-владельца
</ResponseField>

<ResponseField name="list_name" type="string">
  Название списка
</ResponseField>

<ResponseField name="anime_ids" type="array">
  Массив ID аниме в списке
</ResponseField>

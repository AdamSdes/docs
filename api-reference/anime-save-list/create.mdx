---
title: "Создать список аниме"
api: "POST http://207.154.237.52:8000/anime_save_list/create-anime-save-list/{list_name}"
description: "Создание нового списка сохранённых аниме для авторизованного пользователя"
---

## Параметры пути

<ParamField path="list_name" type="string" required>
  Название списка. Допустимые значения: `Watching`, `Completed`, `On Hold`, `Dropped`, `Plan to Watch`
</ParamField>

## Заголовки

<ParamField header="Authorization" type="string" required>
  Bearer-токен для аутентификации
</ParamField>

## Ответ

<ResponseField name="id" type="string">
  UUID созданного списка
</ResponseField>

<ResponseField name="user_id" type="string">
  UUID пользователя-владельца
</ResponseField>

<ResponseField name="list_name" type="string">
  Название списка (`Watching`, `Completed`, `On Hold`, `Dropped`, `Plan to Watch`)
</ResponseField>

<ResponseField name="anime_ids" type="array">
  Массив ID аниме в списке (при создании пустой)
</ResponseField>

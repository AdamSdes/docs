---
title: "Авторизация (Login)"
api: "POST http://207.154.237.52:8000/auth/token"
description: "Получение access и refresh токенов по логину и паролю"
---

## Параметры запроса

<ParamField query="remember_me" type="boolean" default="true">
  Запомнить пользователя (влияет на срок жизни токенов)
</ParamField>

## Тело запроса

<Note>
  Запрос отправляется как `application/x-www-form-urlencoded` (OAuth2 форма).
</Note>

<ParamField body="username" type="string" required>
  Имя пользователя
</ParamField>

<ParamField body="password" type="string" required>
  Пароль пользователя
</ParamField>

<ParamField body="grant_type" type="string">
  Тип гранта (можно оставить пустым)
</ParamField>

<ParamField body="scope" type="string">
  Скоуп (можно оставить пустым)
</ParamField>

<ParamField body="client_id" type="string">
  ID клиента (можно оставить пустым)
</ParamField>

<ParamField body="client_secret" type="string">
  Секрет клиента (можно оставить пустым)
</ParamField>

## Ответ

<ResponseField name="user" type="object">
  Данные пользователя

  <Expandable title="Свойства">
    <ResponseField name="id" type="string">
      UUID пользователя
    </ResponseField>
    <ResponseField name="email" type="string">
      Email
    </ResponseField>
    <ResponseField name="username" type="string">
      Имя пользователя
    </ResponseField>
    <ResponseField name="nickname" type="string">
      Никнейм
    </ResponseField>
    <ResponseField name="status" type="string">
      Роль пользователя
    </ResponseField>
    <ResponseField name="user_avatar" type="string">
      Путь к аватару
    </ResponseField>
    <ResponseField name="user_banner" type="string">
      Путь к баннеру
    </ResponseField>
    <ResponseField name="password" type="string">
      Хешированный пароль
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="access_token" type="string">
  JWT access-токен для авторизации запросов
</ResponseField>

<ResponseField name="refresh_token" type="string">
  JWT refresh-токен для обновления access-токена
</ResponseField>

<ResponseField name="token_type" type="string">
  Тип токена (`bearer`)
</ResponseField>

<RequestExample>
```bash cURL
curl -X 'POST' \
  'http://207.154.237.52:8000/auth/token?remember_me=true' \
  -H 'accept: application/json' \
  -H 'Content-Type: application/x-www-form-urlencoded' \
  -d 'grant_type=&username=whyroma&password=%40Ralef1905&scope=&client_id=&client_secret='
```

```javascript JavaScript
const params = new URLSearchParams();
params.append("username", "whyroma");
params.append("password", "@Ralef1905");

const response = await fetch(
  "http://207.154.237.52:8000/auth/token?remember_me=true",
  {
    method: "POST",
    headers: {
      Accept: "application/json",
      "Content-Type": "application/x-www-form-urlencoded",
    },
    body: params,
  }
);
const result = await response.json();
```

```python Python
import requests

response = requests.post(
    "http://207.154.237.52:8000/auth/token",
    params={"remember_me": True},
    data={
        "username": "whyroma",
        "password": "@Ralef1905"
    }
)
result = response.json()
```

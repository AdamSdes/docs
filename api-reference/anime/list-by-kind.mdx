---
title: "Список аниме по типу"
api: "GET http://207.154.237.52:8000/anime/get-anime-list-by-kind/{kind}"
description: "Получение списка аниме, отфильтрованного по типу (tv, movie, ova и т.д.)"
---

## Параметры пути

<ParamField path="kind" type="string" required>
  Тип аниме. Допустимые значения:

  | Значение | Описание |
  |---|---|
  | `tv` | TV-сериал |
  | `tv_special` | TV-спецвыпуск |
  | `ona` | ONA (онлайн) |
  | `movie` | Полнометражный фильм |
  | `ova` | OVA |
  | `special` | Спецвыпуск |
</ParamField>

## Параметры запроса

<ParamField query="page" type="integer" default="1">
  Номер страницы
</ParamField>

<ParamField query="limit" type="integer" default="1">
  Количество записей на странице
</ParamField>

## Ответ

<ResponseField name="total_count" type="integer">
  Общее количество аниме данного типа
</ResponseField>

<ResponseField name="anime_list" type="array">
  Массив аниме (структура аналогична [GET /anime/get-anime-list](/api-reference/anime/list))
</ResponseField>

<RequestExample>
```bash cURL
curl -X 'GET' \
  'http://207.154.237.52:8000/anime/get-anime-list-by-kind/tv?page=1&limit=1' \
  -H 'accept: application/json'
```

```javascript JavaScript
const response = await fetch(
  "http://207.154.237.52:8000/anime/get-anime-list-by-kind/tv?page=1&limit=1",
  {
    headers: { Accept: "application/json" },
  }
);
const result = await response.json();
```

```python Python
import requests

response = requests.get(
    "http://207.154.237.52:8000/anime/get-anime-list-by-kind/tv",
    params={"page": 1, "limit": 1},
    headers={"Accept": "application/json"}
)
result = response.json()
```

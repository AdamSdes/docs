---
title: "Список аниме по рейтингу"
api: "GET http://207.154.237.52:8000/anime/get-anime-list-by-rating/{rating}"
description: "Получение списка аниме, отфильтрованного по возрастному рейтингу"
---

## Параметры пути

<ParamField path="rating" type="string" required>
  Возрастной рейтинг. Допустимые значения:

  | Значение | Описание |
  |---|---|
  | `g` | Все возрасты |
  | `pg` | Дети |
  | `pg_13` | Подростки 13+ |
  | `r` | 17+ (насилие и нецензурная лексика) |
  | `r_plus` | 17+ (лёгкая эротика) |
</ParamField>

## Параметры запроса

<ParamField query="page" type="integer" default="1">
  Номер страницы
</ParamField>

<ParamField query="limit" type="integer" default="1">
  Количество записей на странице
</ParamField>

## Ответ

<ResponseField name="total_count" type="integer">
  Общее количество аниме с данным рейтингом
</ResponseField>

<ResponseField name="anime_list" type="array">
  Массив аниме (структура аналогична [GET /anime/get-anime-list](/api-reference/anime/list))
</ResponseField>

<RequestExample>
```bash cURL
curl -X 'GET' \
  'http://207.154.237.52:8000/anime/get-anime-list-by-rating/pg_13?page=1&limit=1' \
  -H 'accept: application/json'
```


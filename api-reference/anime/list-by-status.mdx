---
title: "Список аниме по статусу"
api: "GET http://207.154.237.52:8000/anime/get-anime-list-by-status/{status}"
description: "Получение списка аниме, отфильтрованного по статусу выхода"
---

## Параметры пути

<ParamField path="status" type="string" required>
  Статус аниме. Допустимые значения:

  | Значение | Описание |
  |---|---|
  | `anons` | Анонсировано |
  | `ongoing` | Выходит |
  | `released` | Завершено |
</ParamField>

## Параметры запроса

<ParamField query="page" type="integer" default="1">
  Номер страницы
</ParamField>

<ParamField query="limit" type="integer" default="1">
  Количество записей на странице
</ParamField>

## Ответ

<ResponseField name="total_count" type="integer">
  Общее количество аниме с данным статусом
</ResponseField>

<ResponseField name="anime_list" type="array">
  Массив аниме (структура аналогична [GET /anime/get-anime-list](/api-reference/anime/list))
</ResponseField>

<RequestExample>
```bash cURL
curl -X 'GET' \
  'http://207.154.237.52:8000/anime/get-anime-list-by-status/anons?page=1&limit=1' \
  -H 'accept: application/json'
```

```javascript JavaScript
const response = await fetch(
  "http://207.154.237.52:8000/anime/get-anime-list-by-status/anons?page=1&limit=1",
  {
    headers: { Accept: "application/json" },
  }
);
const result = await response.json();
```

```python Python
import requests

response = requests.get(
    "http://207.154.237.52:8000/anime/get-anime-list-by-status/anons",
    params={"page": 1, "limit": 1},
    headers={"Accept": "application/json"}
)
result = response.json()
```
</RequestExample>

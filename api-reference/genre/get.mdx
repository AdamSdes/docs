---
title: "Получить жанр по ID"
api: "GET http://207.154.237.52:8000/genre/get-genre/{genre_id}"
description: "Получение информации о жанре по его Shikimori ID"
---

## Параметры пути

<ParamField path="genre_id" type="string" required>
  ID жанра (Shikimori ID, например `2`)
</ParamField>

## Ответ

<ResponseField name="id" type="string">
  Внутренний UUID жанра
</ResponseField>

<ResponseField name="genre_id" type="string">
  ID жанра (Shikimori)
</ResponseField>

<ResponseField name="name" type="string">
  Английское название жанра
</ResponseField>

<ResponseField name="russian" type="string">
  Русское название жанра
</ResponseField>

<RequestExample>
```bash cURL
curl -X 'GET' \
  'http://207.154.237.52:8000/genre/get-genre/2' \
  -H 'accept: application/json'
```

```javascript JavaScript
const response = await fetch(
  "http://207.154.237.52:8000/genre/get-genre/2",
  {
    headers: { Accept: "application/json" },
  }
);
const genre = await response.json();
```

```python Python
import requests

response = requests.get(
    "http://207.154.237.52:8000/genre/get-genre/2",
    headers={"Accept": "application/json"}
)
genre = response.json()
```
</RequestExample>

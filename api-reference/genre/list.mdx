title: "Список жанров"

description: "Получение полного списка всех жанров аниме"
---

## Ответ

Возвращает массив объектов жанров.

<ResponseField name="id" type="string">
  Внутренний UUID жанра
</ResponseField>

<ResponseField name="genre_id" type="string">
  ID жанра (Shikimori)
</ResponseField>

<ResponseField name="name" type="string">
  Английское название жанра
</ResponseField>

<ResponseField name="russian" type="string">
  Русское название жанра
</ResponseField>

<RequestExample>
```bash cURL
curl -X 'GET' \
  'http://207.154.237.52:8000/genre/get-list-genres' \
  -H 'accept: application/json'
```

```javascript JavaScript
const response = await fetch(
  "http://207.154.237.52:8000/genre/get-list-genres",
  {
    headers: { Accept: "application/json" },
  }
);
const genres = await response.json();
```

```python Python
import requests

response = requests.get(
    "http://207.154.237.52:8000/genre/get-list-genres",
    headers={"Accept": "application/json"}
)
genres = response.json()
```
</RequestExample>

## Button

Базовый компонент кнопки для взаимодействия с интерфейсом.

### Спецификация

| Свойство         | Значение                  |
|------------------|---------------------------|
| Высота           | `50px`                    |
| Padding X        | `12px` / `20px` (sm+)     |
| Padding Y        | `14px`                    |
| Фон              | `#c8c7ca`                 |
| Фон (hover)      | `#b3b3b3`                 |
| Border Radius    | `12px` (`rounded-xl`)     |
| Transition       | `colors`                  |
| Gap (иконка)     | `10px`                    |

### Tailwind CSS

```css
h-[50px] px-3 sm:px-5 py-3.5 bg-[#c8c7ca] rounded-xl flex justify-center items-center gap-2.5 cursor-pointer hover:bg-[#b3b3b3] transition-colors
```

### React / Next.js

```tsx
interface ButtonProps {
  children: React.ReactNode;
  onClick?: () => void;
  icon?: React.ReactNode;
  disabled?: boolean;
  className?: string;
}

export function Button({ children, onClick, icon, disabled, className }: ButtonProps) {
  return (
    <button
      onClick={onClick}
      disabled={disabled}
      className={`h-[50px] px-3 sm:px-5 py-3.5 bg-[#c8c7ca] rounded-xl flex justify-center items-center gap-2.5 cursor-pointer hover:bg-[#b3b3b3] transition-colors disabled:opacity-50 disabled:cursor-not-allowed ${className ?? ""}`}
    >
      {icon && <span>{icon}</span>}
      {children}
    </button>
  );
}
```

### Использование

```tsx
// Базовая кнопка
<Button>Смотреть</Button>

// С иконкой
<Button icon={<PlayIcon />}>Смотреть</Button>

// С обработчиком
<Button onClick={() => console.log("click")}>Войти</Button>

// Отключённая
<Button disabled>Недоступно</Button>
```
